## Project Overview
This repository contains my Digital Literacy Portfolio, developed as part of the Digital Literacy course.

As a Student Digital Ambassador, this project focuses on spreading awareness about the effective and responsible use of digital tools. It includes practical tasks such as creating an infographic, building professional profiles, exploring digital platforms, practicing email etiquette, and understanding cyber safety.

This project reflects my ability to use digital tools efficiently, communicate professionally, and promote safe online practices.
